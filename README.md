# Midnight Bites 

*Midnight Bites* is a multi-page HTML website designed for a late-night café.  
It showcases the café menu,  reviews and contact details.  

This project is built using *pure HTML only* (No CSS, No JavaScript) and is suitable for beginners learning *Git, GitHub, and web project structure*.

---

## Features

- Home page with café introduction  
- Menu page with images and item names  
- Specials page with daily special dishes  
- Gallery page with café and food images  
- Events page with live café events  
- Team page introducing staff and chefs  
- Reviews page with customer feedback  
- FAQ page for common queries  
- Contact page with address, opening hours, and contact form  
- Clean folder structure with proper file naming
